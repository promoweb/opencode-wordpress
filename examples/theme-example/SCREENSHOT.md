# Screenshot Guidelines

## Theme Screenshot Requirements

For WordPress.org theme submission, you need a screenshot.png file.

### Specifications
- **Filename**: `screenshot.png`
- **Dimensions**: 1200 x 900 pixels (recommended)
- **Format**: PNG
- **Location**: Theme root directory
- **File Size**: Keep under 1MB for fast loading

### Screenshot Creation Steps

1. **Set Up a Demo Site**
   - Install WordPress
   - Add sample content (Theme Unit Test Data)
   - Configure theme with best appearance
   - Add widgets to all widget areas
   - Create menus

2. **Capture the Screenshot**
   - Use browser at 1200x900 resolution
   - Take screenshot of homepage
   - Show header, content, sidebar, footer
   - Ensure widgets are visible
   - Show custom logo if available

3. **Edit the Screenshot**
   - Use image editor (Photoshop, GIMP, Canva)
   - Resize to exactly 1200x900 pixels
   - Optimize file size (< 1MB)
   - Save as PNG format

### Screenshot Best Practices

✅ **DO:**
- Show the theme in its best light
- Include header, content, sidebar, footer
- Use high-quality images
- Display custom logo
- Show widgets in action
- Use engaging, colorful content

❌ **DON'T:**
- Use low-resolution images
- Show empty widget areas
- Include personal/sensitive information
- Use placeholder text
- Show admin bar
- Include browser chrome

### Example Screenshot Content

**Header**
- Custom logo
- Site title and description
- Primary navigation menu

**Main Content Area**
- Featured post with image
- Post list with thumbnails
- Proper typography showcase

**Sidebar**
- Search widget
- Recent posts
- Categories
- Custom widgets

**Footer**
- Footer widgets (3 columns)
- Footer menu
- Copyright info

### Tools for Creating Screenshots

1. **Browser Developer Tools**
   - Chrome DevTools device mode
   - Set to 1200x900 resolution

2. **Screenshot Tools**
   - macOS: Screenshot (Cmd+Shift+4)
   - Windows: Snipping Tool
   - Browser extensions: Awesome Screenshot

3. **Image Editors**
   - Adobe Photoshop
   - GIMP (free)
   - Canva (online)
   - Photopea (online, free)

### Testing Your Screenshot

1. Place `screenshot.png` in theme root
2. Go to Appearance > Themes
3. Verify screenshot displays correctly
4. Check file size is under 1MB

### Theme Unit Test Data

For realistic content, use WordPress Theme Unit Test Data:
https://codex.wordpress.org/Theme_Unit_Test

This provides:
- Various post formats
- Comments
- Categories and tags
- Pages with different templates
- Media attachments
