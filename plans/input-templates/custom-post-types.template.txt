# Custom Post Types Template

## Example: Portfolio
Post Type Key: portfolio
Name: Portfolio Items
Singular Name: Portfolio Item
Description: Portfolio and project showcase
Public: true
Has Archive: true
Menu Icon: dashicons-art
Supports:
- title
- editor
- thumbnail
- excerpt
- custom-fields
Taxonomies:
- portfolio_category (categories)
- portfolio_tag (tags)

## Notes
- Post Type Key: Must be unique, lowercase, no spaces (max 20 characters)
- Name: Plural display name
- Singular Name: Single item display name
- Public: Whether visible in admin and frontend
- Has Archive: Whether to have archive page
- Menu Icon: WordPress dashicon or custom icon URL
- Supports: Features the CPT supports
- Taxonomies: Custom taxonomies to register
